# 🎉 ¡Ponme debajo de tu cintura!

Página estática juguetona con un botón sorpresa. Al hacer click aparece una foto con confeti y animaciones.

## Ver online
Publicada con GitHub Pages: https://jorchvr.github.io/ethernet/

## Archivos
- `index.html` — estructura
- `style.css` — estilos y animaciones
- `script.js` — botón mágico, confeti y esquive
- `malcolm.png` — la sorpresa

## Correr local
Solo abre `index.html` en tu navegador.
